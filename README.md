## ⚡ Features

- 💊 **Generic vs Branded Comparison** — Compare medicines by composition, price, and effectiveness
- 🔍 **Medicine Search** — Search any medicine by name, salt, or composition
- 💰 **Cost Savings Calculator** — See how much you save by switching to generic alternatives
- 📋 **Drug Information** — Detailed info on uses, side effects, and dosage
- 🏪 **Pharmacy Integration** — Find where medicines are available
- 🤖 **AI-Powered Recommendations** — Smart suggestions for generic alternatives

## 🚀 Getting Started

### Prerequisites
- Node.js 16+
- npm

### Frontend Setup
```bash
cd client
npm install
npm start
```

### Backend Setup
```bash
cd server
npm install
npm start
```

The app will be available at `http://localhost:3000`

## 💡 Why MED AI?

In India, branded medicines can cost **5-10x more** than their generic equivalents with the **same active ingredients**. MED AI bridges this information gap, empowering patients to make informed, affordable healthcare decisions.
